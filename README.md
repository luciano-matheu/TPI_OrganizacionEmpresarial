Trabajo Practico Integrador de Organización Empresarial
# Integrantes

# Descripción 
El programa simula un pedido de vacaciones de un empleado, basandose el saldo de dias, la fecha y la aprobación de RRHH.
# Instrucciones de uso
Para la experiencia completa usar el legajo N° 005 perteneciente a Diego Ramírez con un saldo de 12 dias para sus vacaciones.
